Overview

The FRC 2010 software includes three separate programming environments: LabVIEW™, Wind River® Workbench, and the Sun™ SPOT Java™ Development Kit for FRC. Use any of these environments to develop the robotics program you want to run on the CompactRIO device. Use LabVIEW to program a robot in the LabVIEW graphical programming environment or to develop a desktop application. Use Wind River Workbench to program a robot in C or C++. Use the Sun SPOT JDK to program a robot in Java.

New Features and Changes

Refer to the FIRST Web site at www.usfirst.org/frccontrolsystem for up-to-date information about new features and changes in all three programming environments in the FIRST Robotics Competition 2010 software.

System Requirements

LabVIEW supports Windows Vista/XP/2000.

Wind River Workbench supports Windows XP Professional for (32-bit), Service Pack 2 and Windows Vista Business for (32-bit), Service Pack 1.

Sun SPOT JDK supports Windows Vista/XP/2000.

If you use LabVIEW to program a robot, refer to the LabVIEW Release Notes, available by navigating to the National Instruments\LabVIEW 8.6\manuals directory and opening LV_Release_Notes.pdf, for a complete list of system requirements.

If you use C or C++ to program a robot, refer to the Wind River FRC Readme, available by navigating to the WindRiver directory and opening readme.txt. Refer to this readme for information about the Wind River software.

Installation and Activation Instructions

Refer to the FIRST Robotics Competition Software Release Notes for information about installing and activating the FRC software. This printed document is part of the FRC software kit.

After you install the FRC software from the FIRST Robotics Competition Software DVDs, you MUST download and install the mandatory FRC software update from the FIRST Web site at www.usfirst.org/frccontrolsystem. You do not need to install previous FRC software updates first.

Accessing the Help

The following documents contain information that you may find helpful as you use the FRC software.

If you use LabVIEW to program the robot, refer to the following documents:

  • Getting Started with LabVIEW for the FIRST Robotics Competition—Use this manual to learn about the LabVIEW graphical programming environment and the basic LabVIEW features you can use to build FRC applications. Access this manual by navigating to the National Instruments\LabVIEW 8.6\manuals directory and opening FRC_Getting_Started.pdf. You also can access this manual from the Getting Started window.
  • LabVIEW Robotics Programming Guide for the FIRST Robotics Competition—Use this manual to learn robotics programming concepts, reference information about FIRST Robotics Competition VIs, and guidelines for troubleshooting in LabVIEW. Access this manual by navigating to the National Instruments\LabVIEW 8.6\manuals directory and opening FRC_Programming_Guide.pdf. You also can access this manual from the Getting Started window.
  • LabVIEW Help—Use the LabVIEW Help to access information about LabVIEW programming concepts, step-by-step instructions, and reference information. The LabVIEW Help also contains information about the FIRST Robotics Competition VIs and dialog boxes. Access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

If you use Wind River Workbench to program the robot, refer to the following documents to get started building an application for FRC:

  • C/C++ Programming Guide for the FIRST Robotics Competition—Use this manual to learn robotics programming concepts, reference information about the WPI Robotics Library, and guidelines for troubleshooting in Wind River Workbench. Access this manual by navigating to the WindRiver\docs\extensions\FRC directory and opening C Programming Guide for FRC.pdf.
  • Wind River Workbench User's Guide—Use the Wind River Workbench User's Guide to learn about building and debugging an application in the Wind River programming environment. Access the Wind River Workbench User's Guide by selecting Help»Help Contents in Wind River and navigating to the Wind River Documentation»Guides»Host Tools»Wind River Workbench User's Guide 3.0, VxWorks Version book on the Contents tab of the Help – Wind River Workbench help window.

If you use the Sun SPOT JDK to program the robot, refer to the following documents to get started building an application for FRC:

  • Release Notes for Java Programming—Use these release notes to access information about installing the WPILib Sun SPOT JDK extensions you can use to program a robot in Java. Access these release notes by navigating to the FRCJava directory and opening Release notes for Java programming.pdf.
  • C and Java Programming Guide for FRC—Use this guide to learn robotics programming concepts in Java, reference information about the WPI Robotics Library, and guidelines for troubleshooting in Java. Access this guide by navigating to the FRCJava directory and opening C and Java Programming Guide for FRC.pdf.

Documentation Updates and Additions

The following items are corrections to the LabVIEW documentation for the FIRST Robotics Competition:

  • The Add to the block diagram and Find on the palette buttons in the Read VI help topic do not add or find the correct VI. You can find the Read VI by navigating to the Functions»WPI Robotics Library»DriverStation»Network Variable palette.
  • The Variable Name input of the Create VI and the Subscribe VI is always a required input.
  • The description of the Variable Reference Out output of the Read VI and the Update VI should read Variable Reference Out.
